And after reboot:

adb install Magisk.apk

Get boot.img:
- Load TWRP (new recovery now) to access as root to all files

adb shell
dd if=/dev/block/bootdevice/by-name/boot of=/sdcard/boot.img

Save it on PC in case of failure.
adb pull /sdcard/boot.img stock_boot.img

Patch boot.img with magisk
Boot system
load magisk app and press install
select previously saver boot.img
Check where magisk save the patched image

get it on the pc with 
adb pull /storage/emulated/0/Download/magisk_patched-27000_eZfiK.img .

adb reboot bootloader

Flash patched image:
sudo fastboot flash boot magisk_patched-27000_eZfiK.img
